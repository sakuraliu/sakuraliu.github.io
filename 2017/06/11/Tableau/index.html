Tableau

发表于 2017-06-11 | 分类于 笔记

Tableau

数据

现在“大数据”是一个热词，但是对于大数据到底是什么，我一直半懂不懂，接触到的也大多是抽象性的描述，今天实际操作Tableau，却真切的被数据的力量
震撼。

可视化

之前学习过Dashboard跟Excel的可视化操作，都还在对于数据本身的处理范围内，而且数据也相对简单。而在Tableau中，数据处理的逻辑完全不同。
数据的具体的数值不重要，重要的是数值之间的关系和内容，通过可视化的操作，大量的繁琐的操作被省略，要做的事情就是把数据当作积木使用，不用再跟
函数打交道，所要做的就是把各个字段进行组合，然后直接绘制数据，没错，就是绘制，简单、直接、有效。

故事

Tableau提供了故事的功能，简单的说就是提供对数据进行叙述的功能，听起来这像是PPT+Excel+Onenote，实际上却远远不止，近乎一站式的解决了数据的问题，无论在个人的数据处理还是团队协作上，都提供了相当有效率的方案。

技术

技术的飞快进步和知识的迅速迭代，是对于现在人的巨大挑战，人力在日益发展的技术面前显得微不足道，更确切的说大数据、云技术这些技术不仅仅是一个工具，更像是一个新的次元，打破了认识的边界，带来了根本性的变革。

与技术为友

技术的迭代是大势，个人甚至国家在这浪潮中都显得无能为力，虽然之前有声音认为技术已经进入瓶颈期，但是就亚马逊、谷歌等新技术公司的发展，技术的瓶颈似乎在以另一种方式被打破，与其被技术裹挟，不如与技术为友，拥抱改变带来的挑战。
