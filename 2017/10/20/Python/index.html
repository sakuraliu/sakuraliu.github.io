Python扫雷日记（一）

发表于 2017-10-20 | 分类于 Note

环境

  • 系 统：macOS Sierra
  • Xcode: version 9.0

    安装和使用Python

    安装

    安装Python有多种方法，下面进行简单介绍。
  1. 直接官网下载
    双击直接解压安装，截止到10.16，最新的版本为3.6.2，附带IDLE编辑器。
  2. 在终端使用homebrew命令安装

    brew install Python

附：
Homebrew 是由Max Howell开发的软件包管理系统，使用非常方便，安装的方法是在终端输入:

ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"a

常用的命令有：

brew install 软件名 %安装软件
brew search 软件名 % 搜索软件
brew uninstall 软件名 %卸载软件
brew update %更新所有软件

使用

作为初学者，并没有太多实际性的建议，提供几种可以选择的Python使用方法。

  1. PyCharm
    PyCharm 是一个大牛专门为Python设计的Python编辑器,该编辑器有三个版本，Professional、Free Community 和Edu版，最令人惊喜的是，支持对于学生的该网站的全系列Jetbrain工具箱免费，使用邮箱激活一次，可以使用一年，激活的方式是使用自己的edu邮箱到官网的位置，输入自己 的edu邮箱和地区，Jetbrain会发两封邮件，一封点击后会再发一封，第二封点开会有该网站的注册链接，直接注册账号加上复杂密码就可以，在期限内，全系列都可以凭借账号和密码激活，强烈推荐。
  2. Xcode+Python
    Xcode在Mac里是免费的编辑器，集成了多种语言，其中也自带了Python，但是自带的Python 是2.7.1版本，自08年版本迭代以来，Python3.x逐渐成为主流，因而对于初学者最好从Python3.x开始。对于Xcode+Python需要进行设置。
  3. Sublime Text +Python
    Sublime是广受好评的一个文本编辑器，配色好看，轻量化，支持多种语言，跨多平台，也是个很优秀的选择。
  4. IDLE
    直接在附带下载的编辑器内编辑。

    雷区

    Python版本

    mac当下的系统自带的Python是2,7.1版本，而现在的Python3.x需要自行下载，下载后，会出现版本冲突的问题。因为学习的需要，我下载了Python3.x版本，但是打开IDLE编辑器出现：

    WARNING: The version of Tcl/Tk (8.5.9) in use may be unstable.
    Visit http://www.python.org/download/mac/tcltk/ for current information.

Google后使用homebrew升级Tcl/Tk，但是升级的是系统自带的2.7.1的Tcl/Tk，导致依旧无法输入中文。

  • 单行代码
  • 程序运行
